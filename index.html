Add Content

First write out the <html></html> tag. Between the <html></html> tag add the <head></head> tag. In the head tag you can add <meta></meta> tags giving information about the website such as the author and description, <title></title> tag to give your website a title which the users can see on the browser tab, and <link></link> tags to link css stylesheets and CDNs. To add content to the website you need to create a <body></body> tag between the <html></html> tag. Here you can add other tags such as <p></p> to add text such as this.

Use different tags to represent different content.

You can use different tags to represent the different content you are adding to the body. <h1></h1> can give text an important look for titles or articles, <code></code> tags are used to include code snippets as content, and <a></a> tags are used for hyperlinks, and etc. There are many, many more tags which you can check out at www.html-5-tutorial.com/all-html-tags.htm.

Add Spacing and Use Whitespace

Spacing and whitespace is one of the most fundamental building blocks to designing a good website. The spacing on a website makes it easier for the users to scan the contents of your website and increases the readability of the content's text. By using the whitespace of a website you can control the users attention to certain elements and create a balance throughout your website.

Use containers, margins, padding and borders.

One of the main ways to create whitespace and tell the users where the main content resides is through the use of containers. Using containers with css such as flexbox you can control how the main content will be placed and gives a much cleaner look. The three main ways to create spacing on html elements are margins paddings and borders. Margins are used in websites to create space between html elements. This is to avoid clutter and improve visibility. Padding is used to create space between the container element and the content inside. Borders can be used to provide visible confirmation to the users on how big and where the html element is. For borders, padding is used to create the distance between the content and box borders in css.

Here are some examples of creating whitespace.

Choose a Typeface

Choosing the right typeface is an important task. It is one of the main ways a website can give a unique impression on the users. There are so many different types of typeface that choosing the right one can be overwhelming. However one of the main way of choosing one, is a typeface that increases the ledgibility of the text for the users. Choosing the fontsize through css is another important way to increase the readibility for the users of the website. Using bigger fonts help readers able to acutally see the content while scanning and highlight important text, such titles and such of a website.

How many fonts should I use?

I would say choosing more than three is usually going overboard. You don't want to overwhelm the users with different kinds of fonts all over the place. Fonts have a purpose and usually using more than three ruins the website's cohesive feel. I recommend only using more than three fonts if each font has a specific purpose on the website.

Where can I get some fonts?

Some of the best fonts are available free. The one we choose for this website was found through Google Fonts. This typeface is called Roboto which is the default font that Google uses across their apps. It is easy to read and very legible.

Adding the font and changing the font-size.

custom.css


    body {
        font-family: 'Roboto', sans-serif;
    }

    p {
        font-size: 20px;
    }

    h3 {
        font-size: 30px;
    }
